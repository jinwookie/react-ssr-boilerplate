# React Boilerplate

Boilerplate for React apps

## Included

Packages:
- react >16
- webpack >4
- babel >7
- webpack-serve
- sass
- node/express/pug
- nodemon

Features:
- hot reloading in dev mode
- prod minification
- redux
- react router (for SPA)
- SSR (with data fetching)

## Add-ons needed
- multi-page serves (for non-SPI)
- js linting
- sass linting
- autoprefixing
- unit testing
